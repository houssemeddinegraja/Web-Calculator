# Web-Calculator
HTML/CSS/JS two-number calculator 

**Current available operations:**<br>
* Addition
* Subtraction
* Multiplication
* Division
* Greatest Common Divisor *(in progress)*
* Least Common Multiple *(in progress)*

Design is still to be improved, and final calculator will be published as soon as all elements are finalized.
